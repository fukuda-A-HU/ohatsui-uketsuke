# おはツイキーホルダー 画像受付フォーム

キーホルダー用の画像を、購入者がスマホ／PCから簡単にアップロードできる受付フォームです。
`index.html` を Web に置くだけで動作します（画像の切り抜き調整はブラウザ内で完結）。

## 機能

- **表面・裏面 2枚**の画像をそれぞれアップロード（両面とも必須）
- **印刷方法（フチあり／ふちなし）を購入者が選択**でき、選択内容も送信されます
- ドラッグ＆スライダーで仕上がりサイズ（縦向き 54×72mm）に合わせてトリミング調整
- **完成見本を選択モードでリアルタイム表示** — フチあり＝白フチ付き／ふちなし＝端まで印刷、を各面ごとに確認できます
- 表・裏を 300dpi PNG（縦向き 638×850px）で書き出し、注文情報・印刷方法とともに送信

## セットアップ

1. `gas/コード.gs` の手順に従い Google Apps Script をウェブアプリとしてデプロイします
   （表面 `imageFront` / 裏面 `imageBack` の2枚を受け取り Google Drive に保存します）。
2. 発行された `/exec` URL を `index.html` 冒頭の `ENDPOINT_URL` に貼り付けます。
3. `index.html` を Web ホスティング（GitHub Pages 等）に置いて公開します。

## ファイル構成

- `index.html` — 受付フォーム本体（画像選択・トリミング・完成見本・送信）
- `gas/コード.gs` — 送信先の Google Apps Script（2枚受信 → Drive 保存）
