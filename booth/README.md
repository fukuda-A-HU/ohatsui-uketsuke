# Booth 商品ページ改善案

Booth ショップ [a-kun-lab](https://a-kun-lab.booth.pm/) の商品「【NFC機能付き！】おはツイキーホルダー」
（https://a-kun-lab.booth.pm/items/8560048 ）の商品ページを見直すための提案一式です。
実際の商品ページやリポジトリ本体（`index.html` / `gas/`）には影響しません。

## ファイル構成

- `description.md` — Booth の商品説明欄にそのまま貼り付けられる本文テキスト
- `preview.html` — Booth の商品ページの雰囲気に寄せた見た目確認用のHTMLモック

## 使い方

1. **本文を反映する場合**：`description.md` の内容をコピーし、Booth 管理画面の商品編集
   → 「商品の説明」欄にそのまま貼り付けてください（Markdown記法ではなく、絵文字と罫線で
   見出し・区切りを表現したプレーンテキストです）。
2. **見た目を確認する場合**：`preview.html` をブラウザで直接開いてください（ダブルクリックで
   OK。サーバー起動は不要です）。外部通信・外部CDNには一切依存していません。
   画像部分は実写真の代わりにSVGのプレースホルダーで表示しています。

## 注意

- ここに置いた内容は「提案」であり、Booth の実ページへの反映は手動で行ってください。
- 商品仕様（サイズ・価格・配送方法など）を変更した場合は、`description.md` と
  `preview.html` の「📐 仕様」欄を両方とも更新し、内容がずれないようにしてください。
